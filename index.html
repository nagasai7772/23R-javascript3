1.Write a Javascript code to add an element to the end of an array using push method ?
Example:[1,2,3]---->[1,2,3,4]
Solution:
var arr=[1,2,3];
arr.push(4);
console.log(arr); //output: [1,2,3,4]

2.Write a Javascript code to remove last element from an array using the pop method ?
Example:[1,2,3,4]-----> [1,2,3]
Solution:
var arr=[1,2,3,4];
arr.pop();
console.log(arr); //output:[1,2,3]

3. Write a Javascript code to add an element to the beginning of an array using the unshift method.?
Example: [2,3,4]--->[1,2,3,4];
Solution:
var arr=[2,3,4];
arr.unshift(1);
console.log(arr); //output :[1,2,3,4]

4.Write a Javascript code to remove the first element from an array using the shift method ?
Example :[1,2,3,4]--->[2,3,4]
Solution:
var arr=[1,2,3,4];
arr.shift();
console.log(arr); //output:[2,3,4]

5.Write a Javascript code to convert an array into a string using the join method ?
Example :[1,2,3]--->"123"
Solution:
var arr=[1,2,3];
console.log(arr.join('')) //output:123;
